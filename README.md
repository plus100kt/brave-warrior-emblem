# Emblem Prisma CRUD

## 실행
```bash
npm install
npm run prisma:generate
npm run prisma:push
npm run prisma:seed
npm run dev
```

## 포함 내용
- Prisma + SQLite
- Member / MemberEmblem 모델
- API Route CRUD
- 전투원 생성 / 수정 / 삭제
- DB 조회 기반 메인 화면

## API
- GET `/api/members`
- POST `/api/members`
- PUT `/api/members/:id`
- DELETE `/api/members/:id`
